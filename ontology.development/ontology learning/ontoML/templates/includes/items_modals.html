Data Quality Development Tool
This is a data quality development tool. Automated Data modeling and data harmonization using Ontology Learning It helps the user in data integration and semantic interoperability of datasets. The tool will help the user automate the process of data modeling using machine learning with ontology engineering. The tool can be integrated into an machine learning operation (MLOps) pipeline or AI development portal or used as a service to run in the background of a DevOps environment.
